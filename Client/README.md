# Client (Ionic React Frontend)

This is a **mobile-friendly React application built with Ionic**. It communicates with the backend server, manages user authentication, and displays dynamic content.

## Features
- Built with **React + Ionic UI Kit**
- Uses Capacitor for optional mobile platform integration (Android/iOS)
- Communication with backend REST API
- State management using React hooks
- Ready to run in browser, and easy to convert to mobile app

## Requirements
- Node.js (version 14+ recommended)
- npm

## Installation & Running (Development Mode)
```bash
npm install
ionic serve
