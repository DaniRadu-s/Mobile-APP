# Server (Koa Backend)

This is a Node.js backend built with **Koa** and **WebSockets**. It provides authentication using JSON Web Tokens (JWT) and real-time communication capabilities.

## Features
- REST API built with `koa-router`
- JSON Web Token (JWT) authentication (`jsonwebtoken`)
- Real-time updates using WebSockets (`ws`)
- CORS enabled for cross-origin communication
- JSON request parsing via `koa-bodyparser`

## Requirements
- Node.js (version 14 or newer recommended)
- npm

## Installation & Running
```bash
npm install
npm start
